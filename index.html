Webhooks
All webhook URLs are set directly in the HTML file. See the WEBHOOK_URLS object at the top of the <script> block.
🔐 Auth Events
user.signup
Create Account submitted → fires with name, email, phone
user.login
Every successful sign-in → fires with name, email
user.email_verified
OTP code confirmed after sign-up
💸 Payment Events
payment.proceed_step1
Next clicked on amount/recipient step
payment.proceed_step2
Confirm & Continue clicked (entering PIN)
payment.sent
PIN confirmed — payment fully processed → fires with recipient, amount, reference
payment.cancelled_step2
User went Back from confirm screen
payment.cancelled_pin
User went Back from PIN screen
🛡 Security Events
pin.wrong_attempt
Wrong PIN entered → fires with attempt count & remaining
pin.locked_out
Account locked after 3 failed PIN attempts
identity.verification_submitted
Identity form submitted after lockout
transaction.flagged
Flag button clicked on a transaction
📊 Activity Events
transaction.viewed
Transaction row clicked to open details
add_money.clicked
Add Money button tapped on dashboard
request_money.clicked
Request button tapped on dashboard
contact.quick_send
Quick-send contact chip clicked
send.transfer_type_changed
Transfer type changed (Instant / Standard / etc)
history.filter_changed
Filter chip clicked in Transactions tab
notifications.viewed
Bell icon clicked
chat.opened / chat.closed
Support chat popup opened or closed
chat.message_sent
Message sent in support chat (typed or quick reply)
📄 How to set URLs
Open index.html in any text editor and find:

const WEBHOOK_URLS = { Paste your n8n URL between the quotes for each event:

'user.signup': 'https://...',
'user.login': 'https://...',
'payment.sent': 'https://...',
Leave as '' to silently skip that event.

Use ▶ Test buttons to fire a test payload to whichever URLs are currently set.
